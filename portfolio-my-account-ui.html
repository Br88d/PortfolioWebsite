Viagogo My Account

case study

viagogo - my account

This was a project for my previous employer. We needed to design a simple easy to use my account section of the website which was responsive. I was heavily involved in the design and front end development of this project.

View website

Mobile first

Fully responsive

This project was quite complex as it had many factors to cater for, we had card payments, address fields, ticket information. Getting down the user experience for this project was key before we could begin with any design and build.

This project was built using LESS as well as utilising object-orientated principles (OOCSS) to keep our code lightweight and easy to maintain and scale.
