CASE STUDY

City Index mobile website

This mobile website was designed and built for my current employer City Index. I was the design and development lead for this project. Before this website was released in April 2015 City Index had no mobile presence. So it was a priority for this website to be created as there was a massive target audience which we were not communicating to.

This website was designed and developed with a whole new style and code base as the current website is not responsive and is very dated. We wanted to create a all new smarter code base. This website was developed using SASS which enables me to use multiple smart mixins and variables so changes to this website will be very simple to make in the future.

I introduced font icons into this project which is a self built library of SVG icons which can scale to any size as they are rendered as a font. The logo for instance is a font icon so you can scale the logo to any size and it will always be crisp which is perfect for retina devices.

This website has proved to be a big success and has made a huge increase in conversion. The City Index registration forms are also responsive so customers can create an account in minutes from their mobiles. This website is now being replicated for City Index Australia, Germany and Singapore because of the success it has had in the UK.

Please visit the website and take a full look yourself. You will need to be on a mobile device to view it. cityindex.co.uk
