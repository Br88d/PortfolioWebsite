RESPONSIVE EMAIL TEMPLATES AND WEBSITE

Email Framework

PART ONE

The templates

This was a huge two part project. My current employer were sending out dated non responsive emails which were not working in all email clients and were very unprofessional. So what we wanted to create firstly was a set of responsive email templates, so we could target desktop and mobile devices and maintain a consistent style throughout all emails.

Creating all the templates took some time as firstly we needed to come up with all the elements which we wanted to include in the emails from paragraph blocks to social media elements. Each template and element has gone through vigorous testing and now they work on all email clients and browsers down to ie7.

View a template

PART TWO

The framework

The second part of this project was to create a tool which would host all the templates. I was the lead designer and sole developer to create this tool and all the templates. The internal MVC application allows you to go in select a responsive email template depending on language and download it. Once downloaded you have the ability to use the tools pre built elements and add them to your email if you need very easily.

The tool also has the ability to change languages, When you change language on the homepage all the templates change all of the set content to display the correct contact information, risk warning, header and footer with all appropriate links so the user doesnâ€™t have to add these in each time they create an email.

This tool has been a huge succes and is used thought the company globally it maintains consistency throughout out all regions and ensures that no broken unprofessional emails are being sent out.
