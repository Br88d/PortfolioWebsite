FREELANCE PLUMBING & HEATING WEBSITE

Watermark Plumbing

Case study

Watermark website

Watermark are a Kent based plumbing and heating company they previously had no online presence and they were falling behind local competitors because of this. The idea here was to create a clean easy to use website which would be suitable for all age groups.

View website

Mobile first

Simple and easy to use

Watermark now have a three month waiting list for any new clients because of their new online presence.

The simple design clean design of this website makes it easy for the user to get to where they want and find out all the relative information. The gallery section offer a range of videos which Watermark wanted to have on show, this allows customers to previous work they have done.

The list of services was something which I think stand out really well and is a lot more prominent than just having a list of bullet points. This way it is definitely more inviting.

This website is all custom designed and hand coded to ensure that the best practices were used. It was developed using SASS so if the client would like to make any changes to the website in the future it would be a really simple task. Please take a look at the website for yourself.
